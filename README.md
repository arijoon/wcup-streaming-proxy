# World cup streaming proxy

## How to run

Make sure you have npm installed, then 

### clone the repo

```sh
git clone https://github.com/arijoon/wcup-streaming-proxy.git wcup-stream
cd wcup-stream
```

### run

```sh
npm install
npm run build
npm run server
```

or if you have __yarn__ installed:

```sh
yarn install
yarn build
node server
```
